# Titanic Survival Prediction

## Overview
Built ML classification models to predict passenger survival on the Titanic.

## Tools Used
Python, Scikit-learn, Pandas, Matplotlib, Seaborn

## Results
| Model | Accuracy |
|-------|----------|
| Logistic Regression | 79.89% |
| Random Forest | 82.68% ✅ Best |

## Key Findings
- Females survived significantly more than males
- First class passengers had higher survival rates
- Most important features: Sex, Fare, Age, Pclass
